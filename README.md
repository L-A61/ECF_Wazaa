# Wazaa Immobilier

Site immobilier pour l'entreprise Wazaa Immobilier située à Amiens.

## Installation

Créer la base de donnée afpa_wazaa_immo via le fichier afpa_wazaa_immo.sql présent dans le dossier sql.
